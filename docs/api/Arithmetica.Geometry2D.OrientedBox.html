Remarks

An oriented box is a box whose faces have normals that are all pairwise orthogonal-i.e., it is an axis aligned box arbitrarily rotated. A 2D oriented box is defined by a center point, two orthonormal axes that describe the side directions of the box, and their respective positive half-lengths extents.
