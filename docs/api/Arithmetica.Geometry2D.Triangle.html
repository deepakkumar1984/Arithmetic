Remarks

Given a trianlge represented by P0, P1, P2 where Pi = (xi, yi) the signed area of the triangle is calculated using the following formula: |1 1 1 | Area(P0,P1,P2) = 0.5*det|x0 x1 x2| = 0.5 * ((x1-x0)(y2-y0) - (x2-x0)(y1-y0)) |y0 y1 y2|
