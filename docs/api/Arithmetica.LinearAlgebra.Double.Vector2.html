KrossProduct(Vector2, Vector2)

Calculates the Kross product of two vectors.

Declaration
public static double KrossProduct(Vector2 left, Vector2 right)
Parameters
Vector2 left

A Vector2 instance.
Vector2 right

A Vector2 instance.
Returns
System.Double

The Kross product value.
Remarks

The Kross product is defined as: Kross(u,v) = u.X*v.Y - u.Y*v.X.

The operation is related to the cross product in 3D given by (x0, y0, 0) X (x1, y1, 0) = (0, 0, Kross((x0, y0), (x1, y1))). The operation has the property that Kross(u, v) = -Kross(v, u).
